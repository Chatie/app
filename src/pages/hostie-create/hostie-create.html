What is a Hostie?

Hostie is the home for Botie. In order to make a Botie work, you need a Hostie to run Botie.

More technic:

  1. Hostie is a Wechaty Runtime with CPU & Memory & Network resources.
  2. Botie is a group of Functions(Gifties) that you want your bot behavior.
